எப்படி பயன்படுத்துவது (How to Use)

  1. முதலில் "Start Camera" பொத்தானை அழுத்தி கேமராவை ஆன் செய்யவும்
  2. நீங்கள் உருவாக்கிய QR கோடை கேமராவின் முன்பு பிடிக்கவும்
  3. QR கோட்டில் உள்ள தகவல்கள் தானாகவே காட்டப்படும்
  4. "Save as JSON" பொத்தானை அழுத்தி தகவல்களை சேமிக்கவும்
  5. சேமிக்கப்பட்ட தகவல்கள் JSON format இல் கீழே காட்டப்படும்
  6. "Export JSON" அழுத்தி அனைத்து தகவல்களையும் download செய்யவும்

QR Code Format Example

உங்கள் QR code இல் இந்த format இல் தகவல்கள் இருக்க வேண்டும்:

product: Biscuit, expiry: 20/11/2026
அல்லது
{"product": "Biscuit", "expiry": "2026-11-20"}
